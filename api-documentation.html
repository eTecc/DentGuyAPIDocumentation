Auth API

The auth API returns you a token that is then used in all other API requests.

URL: index.php?controller=auth

POST variables:

Name Value
username string
password string

Return

Name Value
admin_id int
token string

Sending an Auth Token

Each of the following requests require authentication. That must be sent as a HTTP Header called X-Dash-Token.

Reset Password API

URL: index.php?controller=auth&action=reset&username={username}

Return

Name Value
message string

Claims API

List

?controller=claims&filter=(new|unscheduled|scheduled)&per_page=1000

Info

?controller=claims&view=info&claim_id={claim_id}

Add Note

?controller=claims&view=add-note&claim_id={claim_id}&note={note}

A file named “image” can also be posted to this method

Schedule

?controller=claims&view=schedule&claim_id={claim_id}&time={time}&location={location}&notes={notes}

{time} is a Unix timestamp

Unschedule

?controller=claims&view=unschedule&claim_id={claim_id}

Accept

?controller=claims&view=accept&claim_id={claim_id}

Reject

?controller=claims&view=reject&claim_id={claim_id}&reason={reason}

Upload Estimate

?controller=claims&view=upload-estimate

statuses: Array where key is damage ID and value is the status.

Priority Communications API

List

?controller=priority_communications&per_page=1000

Sent Messages

?controller=priority_communications&view=sent&per_page=1000

Mark as Read

?controller=priority_communications&view=mark_as_read&priority_communication_id={priority_communication_id}

Reply

?controller=priority_communications&view=reply&parent_id={parent_id}&comment={comment}

A file named “image” can also be posted to this method

Send

?controller=priority_communications&view=send&claim_id={claim_id}&topic_id={topic_id}&recipient_id={recipient_id}&service={service}&comment={comment}

A file named “image” can also be posted to this method
