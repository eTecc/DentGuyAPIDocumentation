Manual Claim API

Auth API

The auth API returns you a token that is then used in all other API requests.

URL: index.php?controller=auth

POST variables:

Name Value
username string
password string

Return

Name Value
admin_id int
token string

Sending an Auth Token

Each of the following requests require authentication. That must be sent as a HTTP Header called X-Dash-Token.

Claim Info API

URL: index.php?controller=claims&view=info

POST variables:

Name Value
claim_id int

Return

This API returns lots of information about the claim, most of which is irrelevant to your needs. This information is used in our apps. The one variable you need is:

Name Value
current_status string
